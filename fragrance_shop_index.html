Fragrance Cart Builder

This is a single file storefront you can upload to GitHub Pages. Add your own scents and prices, let buyers choose a size and quantity, then export their cart as a PDF with your payment details and GCash QR.

Owner setup: edit STORE_CONFIG and STARTER_CATALOG in the script section. If you want a real GCash QR image, place gcash-qr.png in the same GitHub folder and update the file name below.
